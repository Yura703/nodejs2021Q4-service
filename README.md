## Installation
```bash
$ git clone https://github.com/Yura703/nodejs2021Q4-service.git
```

```bash
$ npm install
```
```bash
$ docker compose up --build
```
```bash
$ npm run start:dev
```

## Test

```bash
$ npm run test:auth
```
Immediately start the database in docker, then run NestJS application.
